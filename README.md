## 주요 기능

  1. 다양한 축제 정보 제공
     - 국내외 다양한 축제 정보를 한눈에 확인할 수 있는 플랫폼
       
  2. 지역 맞춤형 축제 기획서 작성
     - 자신의 지역에 맞는 독창적인 축제 기획서를 작성하고 공유
       
  3. 커뮤니티 기반 소통
     - 다양한 축제 기획서에 댓글로 의견을 나누고 아이디어를 발전.
       
  4. 전문 축제 기획자와의 네트워킹
     - 축제 전문가들과의 연결을 통해 실질적인 조언과 협업 기회 제공.

  <br>
    
  

## 기술 스택

  - 언어 : java
    
  - 프레임워크 : Spring boot
    
  - 버전 컨트롤러 : git, github
    
  - 배포 : Docker, AWS - EC2, ACM, Route53, ALB
    
  - 서버 모니터링 : Sentry
    
  - DB : Mysql, AWS - RDS
